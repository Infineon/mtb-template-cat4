# ModusToolbox CAT4 BSP Template Library Release Notes

### What Changed?
#### v1.1.0
- Added support for BSP Assistant chip flow
- Update MTBX dependencies to use MTB 3.0 Query APIs

Initial release

| GCC Compiler                              | 10.3.1  |
